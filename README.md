Unofficial addon that syncs your Letterboxd data into Stremio.

## Features

- Watchlist & diary
- Private lists
- Friends activity
- Ratings on every film
- Rate & like from Stremio
- Toggle watched & watchlist
- All platforms

## Setup

**Frontend**

```bash
npm install
npm run dev
```

**Backend**

```bash
cd backend
npm install
cp .env.example .env  # fill in your values
npm run dev
```

## Stack

Next.js 16 / React 19 / Tailwind CSS 4 / Fastify 5 / SQLite
